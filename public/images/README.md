# Architecture Diagram Images

Place the following architecture diagram images in this directory:

1. **react-native-flutter-architecture.png**
   - React Native vs Flutter architecture comparison diagram
   - Shows JavaScript/Bridge vs Native Code/Platform Channels
   - Dimensions: Recommended 1200x800px or higher

2. **xamarin-ios-architecture.png**
   - Xamarin iOS architecture overview
   - Shows YourApp.exe, C#/.NET APIs, Binding, iOS APIs, Objective-C Runtime
   - Dimensions: Recommended 1200x800px or higher

3. **cross-platform-architecture.png**
   - 5-Layer Cross-Platform App Architecture
   - Shows User Interface, Business Logic, Data Persistence, Network, Platform Abstraction layers
   - Dimensions: Recommended 1200x800px or higher

## Image Requirements

- **Format**: PNG or JPG
- **Optimization**: Compress images for web (use tools like TinyPNG)
- **Alt Text**: Descriptive alt text is included in blog posts
- **SEO**: Images should have descriptive filenames matching the blog post slugs

## Adding Images

1. Save architecture diagrams as PNG/JPG files
2. Name them exactly as listed above
3. Place in `frontend/public/images/` directory
4. Images will be accessible at `/images/{filename}`
